____________________________________________________________________________________________________
	GNOME - среда рабочего стола.
____________________________________________________________________________________________________
	Устанавливает GNOME
sudo apt-get install gnome-shell
sudo apt-get install gnome-session gdm3

	Устанавливает утилиту для работы с GNOME
sudo apt-get install gnome-tweaks

	Устанавливает утилиту для работы с расширениями
sudo apt install gnome-shell-extension-manager

	Устанавливает связзь с расширением браузера Google Chrome
sudo apt-get install chrome-gnome-shell
____________________________________________________________________________________________________
	Темы для Gnome необходимо хранить в папке
~/.themes

	Иконки и курсор необходимо хранить в папке
~/.icons

	Шрифты необходимо хранить в папке
~/.local/share/fonts
