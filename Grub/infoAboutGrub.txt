	GRUB - загрузчик операционной системы.
___________________________________________________________________________________________________
	Устанавливает Grub
sudo add-apt-repository ppa:danielrichter2007/grub-customizer
sudo apt update
sudo apt install grub-customizer
___________________________________________________________________________________________________
	Чтобы установить тему необходимо зайти в папку с темой и:

	Если имеется файл "install.sh", необходимо его запустить
sudo install.sh


	Если данного файла нет, необходимо сделать следующее:

1. Переместить тему в /boot/grub/themes
sudo cp -r название_папки /boot/grub/themes

2. Запустить тему через приложение Grub Customizer
___________________________________________________________________________________________________
	Можно изменить время ожидания и путь к теме через редактирования файла
/etc/default/grub
sudo nano grub

Устанавливает время
GRUB_TIMEOUT="60"

Устанавливает путь к теме
GRUB_THEME="/boot/grub/themes/virtual_future/theme.txt"
